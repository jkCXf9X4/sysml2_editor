# 02 High-Level Design

This level turns product definition into architecture-level design: what the product must do, which product blocks carry that behavior, and how the major system concepts fit together.

- [functions.sysml](./functions.sysml) defines what the product must do.
- [product-breakdown.sysml](./product-breakdown.sysml) defines the major product blocks and which functions they satisfy.
- [architecture.sysml](./architecture.sysml) models the high-level architecture decisions from `docs/architecture`.

Use this folder when translating product goals into high-level design structure without implementation detail.

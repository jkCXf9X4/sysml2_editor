# PDEC-004: Traceability-First Modeling

Status: Accepted

## Decision

Traceability is a first-class product capability across model items, source files, branches, repositories, and changes.

## Reason

The product vision requires users to follow why a model element exists, where it lives, and what it affects.

## Tradeoffs

The model graph and UI must expose trace links as visible navigation and review facts. Traceability must be recomputable from model and source state rather than hidden in unrelated UI state.

## Applies To

- Model graph
- Inspector and trace views
- Source ownership views
- Diff and impact analysis

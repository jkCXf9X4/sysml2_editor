## Product Summary

`sysml2_editor` is a Git-native SysML v2 architecture workbench.

The product should let users:

- Open a Git repository
- Inspect SysML structure visually and textually
- Edit supported model elements
- Preserve traceability and reviewability
- Write changes back to text safely

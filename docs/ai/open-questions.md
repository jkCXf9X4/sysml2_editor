# Deferred Topics

These are not first-slice blockers, but they should stay visible so they do not get pulled into the wrong phase.

## Deferred Until After The MVP Core

- Desktop packaging
- Native desktop shell decisions
- Full SysML v2 language coverage
- Behavior modeling
- Advanced merge automation
- Rich traceability reporting
- CI-driven report generation

## Deferred Until The Parser And Writer Stabilize

- External parser or language-server integration
- Broader syntax coverage
- Large-scale model library import behavior

## Deferred Until The Visual Core Works

- Floating window polish
- Multi-window orchestration
- Complex layout persistence
- Highly customized lens management
